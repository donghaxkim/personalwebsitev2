# Portfolio3

An extremely minimalist personal portfolio built with React and Tailwind CSS, featuring dark mode support.

## Features

- Clean, minimal design
- Black and white color scheme
- Dark mode toggle (dark grey background with white text)
- Responsive design
- Profile image with circular border
- Built with React and Tailwind CSS

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your profile image as `public/profile.jpg`

3. Edit `src/App.jsx` to update:
   - Your name in the `<h1>` tag
   - Your description in the `<p>` tag

4. Start the development server:
```bash
npm run dev
```

5. Build for production:
```bash
npm run build
```

## Customization

- Update colors in Tailwind classes in `src/App.jsx`
- Modify layout using Tailwind utility classes
- Theme preference is saved in localStorage
